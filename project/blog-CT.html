Công cụ tìm kiếm tốt nhất cho lập trình viên

  • 03/10/2016 /
  • Đinh Công Minh /
  • Lập trình

Microsoft đã hợp tác với HackerRank để tích hợp code snippet vào trang kết quả tìm kiếm từ Bing.

Thông thường, các newbie sẽ dùng đến internet để tìm kiếm các giải thuật và chương trình mẫu. Stackoverflow là một ví dụ điển hình.

Để hỗ trợ các lập trình viên cho việc này, Microsoft đã biến Bing thành một công cụ đặc biệt cho Programmer.

 

Cách sử dụng

 

Bật bing lên bằng bất kỳ trình duyệt nào.

Gõ vào ô tìm kiếm của Bing một thuật toán hoặc một ví dụ bạn cần tìm.

Ở đây tôi thử tìm kiếm thuật toán quick sort cho ngôn ngữ Python trên trình duyệt Edge, từ khóa tôi nhập sẽ là "quicksort in Python"

 

Và bạn sẽ thấy có hẳn nút "RUN CODE" để chạy thử đoạn code vừa tìm được.

Ở góc phải của code snippet này, bạn cũng thấy một Toolbox nhỏ cho phép thay đổi ngôn ngữ lập trình. Có khá nhiều ngôn ngữ được hỗ trợ như Java, C/C++, PHP, Python, Javascript

 

Chú ý:

Nếu bạn không dùng được code snippets, hãy vào phần setting và chuyển region thành US.

Hacker Rank đảm bảo hỗ trợ 80 thuật toán phổ biến cho tính năng Code Snippets này. 

 

 

Còn chần chừ gì nữa, hãy thử Bing ngay thôi..
